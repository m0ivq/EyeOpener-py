# EyeOpener

**EyeOpener** is an experimental binocular vision research platform for standalone VR headsets such as the Meta Quest.

The project aims to provide a flexible environment for exploring binocular vision, image suppression, visual alignment, and stereoscopic visual stimuli using independent per-eye rendering.

EyeOpener is currently implemented using Python, Flask and WebXR, allowing rapid development and experimentation before a future standalone Quest application.

---

## Features

* Independent left and right eye rendering
* Live binocular calibration
* Adjustable image position
* Adjustable image brightness
* WebXR support
* USB development workflow using ADB Reverse
* Modular exercise framework (planned)

---

## Planned Features

* Independent image scaling
* Contrast adjustment
* Blur adjustment
* Colour filters
* Calibration profiles
* Daily exercise programmes
* Session logging and progress tracking
* Standalone Quest application
* Controller-based calibration
* Hand tracking support

---

## Current Status

EyeOpener is currently an early proof-of-concept.

The software can independently render different images to each eye and provides a live calibration environment for experimenting with binocular visual alignment.

Development is ongoing.
